# API Reference

Base URL: `http://localhost:8000` (local) or your deployed backend URL.
Full interactive docs: `GET /docs` (Swagger UI, auto-generated by FastAPI).

## Auth
| Method | Path | Auth | Description |
|---|---|---|---|
| POST | /auth/register | — | Create an account |
| POST | /auth/login | — | OAuth2 form login → access + refresh token |
| POST | /auth/refresh | — | Exchange a refresh token for a new access token |
| POST | /auth/forgot-password | — | Request a password reset token |
| POST | /auth/reset-password | — | Reset password using the emailed token |
| GET | /auth/verify-email | — | Verify email via token |
| GET | /auth/me | Bearer | Current user |

## Prediction
| Method | Path | Auth | Description |
|---|---|---|---|
| POST | /predict | Bearer | multipart form: `file`, `model_name` → prediction + top5 + Grad-CAM |
| GET | /history | Bearer | Paginated prediction history for the current user |
| DELETE | /history/{id} | Bearer | Delete one prediction |

## Models & training
| Method | Path | Auth | Description |
|---|---|---|---|
| GET | /models | — | List registered model versions + metrics |
| POST | /retrain | Bearer | Kick off a background training job, returns a job id |
| GET | /retrain/{id} | — | Poll training status/epoch metrics |

## Dashboard / profile
| Method | Path | Auth | Description |
|---|---|---|---|
| GET | /dashboard | Bearer | Aggregate stats for the current user |
| GET/PUT | /profile | Bearer | Read/update profile |
| PUT | /settings/password | Bearer | Change password |

## Admin (requires `is_admin=true`)
| Method | Path | Description |
|---|---|---|
| GET | /admin/users | List all users |
| PATCH | /admin/users/{id}/disable | Disable a user |
| DELETE | /admin/predictions/{id} | Delete any prediction |
| GET | /admin/analytics | Platform-wide counts |
| GET | /admin/logs | All training run logs |
